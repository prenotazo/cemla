It was done the next:
- Set the reason end of subscription = “E” (Initiatief van leerling of ouders) as suggested (or default) value in the page when the end date is changed to a value "different" to end date of the year school.

All changes are working with the calculated end date of year school (currently is 30/6, after issue #86692 it will be 31/08).

Please run upgrade-2.0.3-002-bug86921.sql and edit "InvalidUnsubscriptionReasonException" in SubscriptionPage_nl.properties.
